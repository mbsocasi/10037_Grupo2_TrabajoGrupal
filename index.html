Descripcion

en la siguiente seccion se mostrara algunos ejemplos del uso de XML, JSON y AJAX.

XML

XML (eXtensible Markup Language) es un lenguaje de marcado diseñado para almacenar y transportar datos estructurados. A diferencia de HTML, se enfoca en describir datos en lugar de presentación. Con una sintaxis basada en etiquetas, XML permite a los usuarios definir sus propias etiquetas y organizar la información jerárquicamente. Es ampliamente utilizado en aplicaciones y tecnologías que requieren intercambio de datos legible y autodescriptivo.

Codigo javascript Para la extraccion de los datos en una tabla

parser = new DOMParser();
xmlDoc = parser.parseFromString(text, "text/xml");

var books = xmlDoc.getElementsByTagName("book");
var table = document.getElementById("bookTable");

for (var i = 0; i < books.length; i++) {
  var title = books[i].getElementsByTagName("title")[0].childNodes[0].nodeValue;
  var author = books[i].getElementsByTagName("author")[0].childNodes[0].nodeValue;

  var row = table.insertRow(i + 1);
  var titleCell = row.insertCell(0);
  var authorCell = row.insertCell(1);

  titleCell.innerHTML = title;
  authorCell.innerHTML = author;
}

JSON

JSON (JavaScript Object Notation) es un formato de intercambio de datos ligero y legible para humanos y máquinas. Se utiliza ampliamente en aplicaciones web y sistemas que requieren enviar, recibir y almacenar datos de manera eficiente. JSON se basa en una estructura de pares clave-valor y se asemeja a la sintaxis de los objetos en JavaScript. Los datos en JSON se representan como objetos, que contienen una colección de pares clave-valor, o como matrices, que contienen una lista ordenada de valores. Esta flexibilidad y simplicidad lo hacen muy popular en la transmisión de datos entre diferentes sistemas y plataformas. JSON es fácil de entender y procesar tanto para los desarrolladores como para las máquinas, lo que lo convierte en un formato ideal para el intercambio de información en aplicaciones web y servicios web.

Codigo javascript para la extraccion

fetch('paises.json')
.then(response => response.json())
.then(data => {
  const countries = data.countries;
  const tabla = document.getElementById("countryTable");
  countries.forEach(country => {
    const name = country.name;
    const population = country.population;
    const territory = country.territory;
    const row = tabla.insertRow();
    const nameCell = row.insertCell();
    const populationCell = row.insertCell();
    const territoryCell = row.insertCell();
    nameCell.innerHTML = name;
    populationCell.innerHTML = population;
    territoryCell.innerHTML = territory;
  });
})
.catch(error => {
  console.error('Error fetching country data:', error);
});

AJAX

AJAX (Asynchronous JavaScript and XML) es una técnica de desarrollo web que permite actualizar partes específicas de una página web sin necesidad de recargarla por completo. AJAX combina varias tecnologías, como JavaScript, XML (aunque en la práctica también se utiliza JSON) y el objeto XMLHttpRequest, para lograr comunicación asíncrona entre el navegador y el servidor. Esto significa que los datos se pueden enviar y recibir en segundo plano, sin interrumpir la experiencia del usuario. AJAX es ampliamente utilizado para crear aplicaciones web interactivas y dinámicas, ya que permite cargar y mostrar contenido nuevo de forma instantánea, realizar validaciones en tiempo real, enviar formularios sin recargar la página y realizar actualizaciones de datos de forma eficiente.

function loadDoc() {
  var xhttp = new XMLHttpRequest();
  xhttp.onreadystatechange = function () {
    if (this.readyState == 4 && this.status == 200) {
      document.getElementById("demo").innerHTML =
        this.responseText;
    }
  };
  xhttp.open("GET", "./ajax_info.txt", true);
  xhttp.send();
}
